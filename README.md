# Proyecto Final Redes III
Red segura multisede con VLAN, firewall, DMZ, VPN, WiFi 802.1X, SD-WAN e IDS/IPS

## Integrantes
- Victor Cubi
- Jordi Marín

## Idea del proyecto
Diseñar e implementar la red de una empresa con:
- sede central
- sucursal remota
- acceso para teletrabajadores
- servicios publicados en DMZ

La propuesta parte de las actividades del curso:
- ACT1: VLAN y topología básica
- ACT2: firewall y políticas
- ACT3: DMZ
- ACT5/ACT6: ataques, hardening y monitorización
- ACT7: IDS/IPS y análisis de tráfico

## Objetivo
Construir una topología de red segura que cumpla los requisitos del enunciado y documentar toda la evolución del proyecto en GitHub.

## Requisitos que vamos a cubrir
- VLAN
- Firewall
- VPN
- Principios CIA
- Control de acceso y autenticación
- WiFi segura
- 802.1X
- SD-WAN

## Extras propuestos
- IDS/IPS
- Análisis con Wireshark
- Medidas frente a Rogue AP / Evil Twin

## Estructura del repositorio
- `docs/` documentación y memoria
- `configs/cisco/` configuraciones de router y switch
- `configs/fortigate/` configuración del firewall
- `diagrams/` topologías
- `evidencias/capturas/` capturas de pantalla
- `evidencias/wireshark/` PCAP y análisis
- `pruebas/` checklist y resultados
